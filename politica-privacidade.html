Política de Privacidade — Papiro

Última atualização: Julho de 2026

1. Filosofia Local-First

O Papiro foi concebido sob o princípio de Privacy by Design. Todos os dados que você insere — textos, registros, configurações, documentos e histórico — são armazenados exclusivamente no banco de dados local do seu dispositivo. Nenhuma informação pessoal é transmitida para servidores externos, nem para o desenvolvedor.

2. Dados Que Coletamos

Nenhum. O desenvolvedor não coleta, não recebe e não armazena qualquer dado seu. Tudo o que você registra permanece somente no seu dispositivo, sob seu controle.

3. Transmissão de Dados

O aplicativo não transmite para fora do seu dispositivo nenhuma informação, incluindo:

  • Nome, endereço, e-mail ou qualquer dado de identificação pessoal;
  • Dados de localização geográfica;
  • Contatos, agenda ou arquivos do dispositivo;
  • Dados biométricos;
  • Dados de saúde, financeiros, itens de estoque ou documentos do Cofre.

Recursos como Financeiro, Estoque e Cofre funcionam inteiramente no seu aparelho: os dados que você digita neles são armazenados e processados apenas localmente e nunca saem do dispositivo.

4. Telemetria e Logs

O Papiro não envia telemetria a servidores externos. Eventuais registros técnicos de depuração existem apenas em memória durante o uso e são descartados ao encerrar o aplicativo.

5. Segurança dos Dados

O banco de dados local é protegido pelos mecanismos de segurança do próprio dispositivo. O desenvolvedor não tem acesso a ele. Recomendamos:

  • Utilizar bloqueio de tela no dispositivo;
  • Manter o sistema operacional e o navegador atualizados;
  • Estar ciente de que, por serem locais, os dados dependem inteiramente do seu dispositivo (ver Seção 8).

6. Direitos do Usuário (LGPD)

O Papiro foi projetado de modo que não realizamos tratamento dos seus dados em servidores próprios — não há um controlador externo retendo suas informações. Por isso, você exerce seus direitos previstos na Lei nº 13.709/2018 diretamente, sem intermediários:

  • Acesso: todos os dados ficam visíveis na própria interface do app, a qualquer momento;
  • Correção: edite ou remova qualquer registro diretamente nas telas do aplicativo;
  • Exclusão: apague os dados locais conforme a Seção 7;
  • Portabilidade: como não mantemos dados sob nossa guarda, não há informação a transferir a terceiros — os dados estão inteiramente sob seu controle, no dispositivo.

7. Exclusão de Dados

Para apagar permanentemente seus dados, limpe o armazenamento do aplicativo no seu dispositivo (em um navegador: apagar os dados do site do Papiro; em um app instalado: limpar o armazenamento do app nas configurações do sistema). Isso destrói de forma definitiva o banco de dados local e todas as preferências. A operação é irreversível e não existe mecanismo de recuperação.

8. Responsabilidade pelos Dados

Como os dados são exclusivamente locais e o aplicativo não oferece backup em nuvem, a preservação das informações é de inteira responsabilidade do usuário. A perda do dispositivo, a desinstalação do aplicativo, a limpeza do armazenamento pelo navegador/sistema ou uma redefinição de fábrica resultará na perda definitiva e irrecuperável de todos os dados.

9. Menores de Idade

O Papiro não transmite nem armazena fora do dispositivo dados de nenhum usuário, inclusive menores de idade. Se você é responsável por uma criança que utiliza o aplicativo, saiba que nenhum dado pessoal deixa o dispositivo.

10. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Recomendamos revisitar este documento a cada atualização do aplicativo. O uso continuado após alterações constitui aceitação da nova versão.

11. Contato

Para exercer seus direitos ou relatar problemas de privacidade, utilize a seção de contato disponível na tela "Sobre" do aplicativo ou no repositório oficial do projeto.
